# ilyaliver.github.io

Персональный сайт (GitHub Pages) пользователя ilyaliver.

Здесь находится статическая HTML-страница(ы), развёрнутая через GitHub Pages и предназначенная для представления личной информации, проектов или портфолио.

## Структура репозитория

- `index.html` — главная страница сайта.
- Остальные статические файлы: CSS, JS, изображения и т.п.

> Примечание: в этом репозитории все файлы в основном представлены в формате HTML.

## Локальный запуск

Так как проект статический, для локальной проверки можно использовать любой простой HTTP-сервер. Например, с помощью Python 3:

```bash
# В корне репозитория
python -m http.server 8000
# затем открыть http://localhost:8000 в браузере
```

Или с помощью Node.js (http-server):

```bash
npm install -g http-server
http-server -p 8000
```

## Развёртывание

Этот репозиторий предназначен для публикации через GitHub Pages. Чтобы сайт был доступен по адресу `https://ilyaliver.github.io`, достаточно сохранить файлы в ветке `main` (или в ветке, указанной в настройках Pages).

## Как внести изменения

1. Клонируйте репозиторий:

```bash
git clone https://github.com/ilyaliver/ilyaliver.github.io.git
```

2. Внесите правки в HTML/CSS/JS.
3. Сделайте коммит и запушьте изменения:

```bash
git add .
git commit -m "Обновить сайт"
git push
```
